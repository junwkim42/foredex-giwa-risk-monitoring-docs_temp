# ForeDex GIWA Risk Monitoring Documentation

Public, bilingual documentation for the **GIWA Ecosystem On-Chain Anomaly Detection System**, prepared by ForeDex for submission to the GASOK program.

## Documentation layout

- `ko/`: Korean source of truth and default GitBook variant
- `en/`: English companion variant (`/en`)
- `DECISIONS.md`: editorial and evidence decisions; not included in GitBook navigation
- `PLACEHOLDERS.md`: image and transaction-evidence handoff list; not included in GitBook navigation

Each language directory is designed to be connected to a separate GitBook Space using its own `README.md` and `SUMMARY.md`.

## Publication model

- GitBook Site: `ForeDex GIWA Risk Monitoring`
- Korean Space: `GIWA Risk Monitoring — 한국어`
- English Space: `GIWA Risk Monitoring — English`
- Korean is the default variant; English uses `/en`.
- Publication remains gated until ForeDex completes the final preview review.

## Source and evidence policy

Korean content is the authoritative source. Dynamic dashboard values must include a capture time or block number. Demo thresholds are illustrative and must not be presented as universal risk standards. This project must not be described as an official GIWA or Upbit product, partnership, endorsement, or listing decision.

# Publication Placeholders

This checklist is excluded from the GitBook navigation. ForeDex will provide or upload the images manually.

Every image should include:

- source
- captured date and time, including timezone
- chain (`GIWA Sepolia`)
- block number when relevant
- `demo-only` label
- Korean and English alternative text
- a caption that distinguishes configuration values from current on-chain state

| ID | Page | Required item | Publication condition |
|---|---|---|---|
| IMG-01 | Home | Full dashboard overview | Replace or remove placeholder |
| IMG-02 | System overview | Data flow or architecture diagram | Mark current demo vs planned expansion |
| IMG-03 | Demo environment | Demo setup overview | Identify test token and GIWA Sepolia |
| IMG-04 | Detection scenarios | Supply mismatch state | Include example threshold and observation time |
| IMG-05 | Detection scenarios | Abnormal mint state | Include related block or transaction link |
| IMG-06 | Detection scenarios | Project-wallet net outflow state | Include monitored wallet and time window |
| IMG-07 | Detection scenarios | Low-liquidity DEX dump state | Include pool and observation window |
| IMG-09 | On-chain verification | Contract page or Read Contract view | Include contract address and block |
| TX-WARNING | On-chain verification | Representative WARNING transaction hash | Resolved with verified outflow and DEX transactions |
| TX-CRITICAL | On-chain verification | Representative CRITICAL transaction hash | Resolved with verified outflow, mint, and DEX transactions |

No unresolved `PLACEHOLDER`, `TODO`, or empty transaction field may remain in the published version. Image placeholders may be replaced or the surrounding draft block removed.
